VSCode_SFTP_설정

 준비물

OS환경 : Windows 10 Pro

  1. VScode Download

  1. 폴더를 생성해서 VSCode를 해당위치에서 열어준다

  2. Extension창에서 SFTP검색 후 설치해준다


  3. F1을 눌러 SFTP: Config를 검색 후 클릭하여 ./vscode/sftp.json을 추가해준다


  4. sftp.json의 설정을 수정해준다(password항목은 추가해서 적어줬음)


  5. 한번 더 F1을 눌러 SFTP:List All을 클릭해준다


  6. SFTP리스트에서 설정으로 추가해준 이름을 클릭해준다


  7. 이제 index.php라는 파일을 해당 폴더내에 추가해준뒤 저장한다


  8. 서버에 파일이 잘 추가되었는지 해당 폴더 위치로 가서 확인해준다


  9. 이제 PHP코드 작성에 도움을 줄 수 있는 Extension을 설치해서 사용하면 PHP개발의 질이 올라간다

PHP Formatter :-> PHP코드 정렬
Format HTML in PHP :-> PHP에서 HTML코드 정렬
PHP Intelephense :-> PHP문법오류 및 설명
PHP IntelliSense :-> PHP문법 자동완성 지원
