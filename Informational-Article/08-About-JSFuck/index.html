    JSFuck란?

    Martin Kleppe이 개발한 JS(JavaScript)를 이용하여 난해하게 프로그래밍한 스타일이라고 합니다.

    해당 [,],(,),!,+ 6글자만으로 JavaScript를 실행시킬 수 있게 만든 것이 JSFuck입니다.

    간단하게 사용해보려면 jsfuck.com에 들어간 후 명령어를 입력하면 JSFuck을 사용해 불 수 있습니다.

    밑에는 몇 가지 단어를 JSFuck으로 치환한 것입니다.

    (![]+[])[+!+[]]                         // "a"
([][+[]]+[])[!+[]+!+[]]                 // "d"
(!+[]+[])[!+[]+!+[]+!+[]]               // "e"
(![]+[])[+[]]                           // "f"
([][+[]]+[])[!+[]+!+[]+!+[]+!+[]+!+[]]  // "i"
(![]+[])[!+[]+!+[]]                     // "l"
([][+[]]+[])[+!+[]]                     // "n"
(!+[]+[])[+!+[]]                        // "r"
(![]+[])[!+[]+!+[]+!+[]]                // "s"
(!+[]+[])[+[]]                          // "t"
([][+[]]+[])[+[]]                       // "u"

    테스트를 해보고 싶다면 F12로 개발자 모드를 킨 다음 (![]+[])[+!+[]]를 입력하면 'a'라는 단어로 바뀌게 됩니다.

    JSFuck로 문자 만들기

    JSFuck방식으로 문자를 만드는 원리를 설명 해 보겠습니다.

    1. JavaScript에서 []가 if문 안에 들어가면 Truthy 이기 때문에 [] 앞에 논리연산자의 NOT기호인 !를 붙혀서 false를 반환 시킬 수 있습니다.

    2. ![]뒤에 []를 더해서 false를 문자열 'false'로 만들 수 있습니다.

    3. 문자열의 배열을 이용해서 배열에 든 'a'를 나타낼 수 있습니다.

    4. 숫자 1을 +!+[]를 이용해서 표현 할 수 있습니다.

    위와 같은 형식을 통해서 만들어 지기 때문에 Javascript에 원래 있는 여러가지 문자들을 이용해서 뽑아내기 때문에 만약 z의 경우 그 문자를 만들기가 쉽지 않아서 엄청 길게 사용할 수 밖에 없기도 합니다.

    왜 ![]에 +[]를 뒤에 붙히는지에 대해 더 자세한 원리를 알고 싶다면 빠리의 택시 운전사블로그를 보시길 바랍니다.

    JSFuck로 실행하기

    그럼이제 JSFuck방식으로 만든 문자를 어떻게 해야 실행 시킬 수 있는지 살펴 보겠습니다.

    해당 방식을 이용해서 'alert(1)'이라는 단어는 만들었지만 해당 단어를 넣어서 실행할 곳이 없습니다.

    그래서 문자열만 넣으면 실행되는 Function인 eval이 필요한데 eval => []["filter"]["constructor"](CODE)() 이렇게 사용하면 CODE에 원하는 단어를 넣어서 사용 할 수 있다고 되어있습니다.

    이유를 알고자 살펴보니 Function이 저기 위치에 있어서 그런 것 같습니다.

    테스트로 []["filter"]["constructor"](CODE)()에 alert(1)을 집어넣으면 알림창이 잘 뜨는 것을 확인 할 수 있습니다.

    하지만 직접 만드는 것은 귀찮아서 jsfuck.com에 들어간 후 Function을 가져왔는데 []["flat"]["constructor"](CODE)()을 사용합니다.

    JSFuck 주의사항

    만약 크롬이 아닌 다른 브라우저 삽입 시에는 개행 문자가 붙어있어서 배열이 한칸 씩 밀려서 c가 아닌 n이 추출 되어 실행되지 않을 수 있기 때문에 해당사항은 알고 써야 합니다.

    그리고 단순한 글자여도 JSFuck로 변환하면 글자가 엄청나게 늘어 날 수도 있기 때문에 글자를 많이 저장 할 수 있는 공간이 있는지 확인해야 하고, 아니면 필터링이 걸리지 않는 다른 문자와 혼합하여 또 다른 JSFuck를 만들어 글자수를 줄여야 합니다.

    JSFuck 사용되는 곳

    XSS에 대한 Filter가 허술 할 때 XSS를 삽입하기 위해 사용 됩니다.

    워게임이나 CTF에서 간혹 이런 방식을 이용한 XSS 공격 문제가 출시 됩니다.
