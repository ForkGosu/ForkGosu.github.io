    정규표현식(Regex)이란?

    정규표현식의 영어는 Regular Expression 이지만 줄여서 보통 RegEx 혹은 RegExp라고 많이 불립니다.

    프로그램을 작성할 때 일정한 규칙을 가진 문자열을 확인하거나 검사 하기위해 정규표현식을 사용합니다.

    유닉스 계열 CLI에서 grep, sed, awk 등의 명령어나 다양한 프로그래밍 환경에서 많이 사용됩니다.

    정규표현식은 엔진에 따라 변경자의 적용 방식이 상이하므로 해당 구현의 매뉴얼을 읽어야 한다.

    예를들어 패턴 변경자 i에 대해 자바스크립트는 /패턴/i 로 쓰지만 파이썬에서는 re.compile(패턴, flags=re.I) 로, Java나 Go에서는 (?i)패턴으로, C#에서는 new Regex(패턴, RegexOptions.IgnoreCase)라고 쓴다.

    사용 예시

    만약 /(http|https|ftp|telnet|news|mms):\/\/[^\"'\s()]+/i라는 자바스크립트의 정규식이 있다면

    아래와 같이 4개의 지역으로 구분됩니다.

    / (http|https|ftp|telnet|news|mms):\/\/[^\"'\s()]+ / i
    패턴 구분자 시작 찾을 문자열의 패턴 패턴 구분자 끝 패턴 변경자(Flag)

    양 끝의 /안에 찾을 문자열의 패턴을 넣고 맨 오른쪽에는 패턴변경자(Flag)를 넣습니다.

    찾을 문자열 패턴

    정규 표현식 정리

    나중에 정규 표현식을 파악하기 쉽게 표 형태로 정리했습니다.

    메타 문자

    메타문자란 프로그래밍에서 예약된 문자정도의 느낌이며 예약된 문자를 쓸때는 바로사용하지 못하고 역슬래시(\)를 붙여 이스케이프 해주면 됩니다.

    표현식 의미
    ^x 문자열의 시작을 표현하며 x 문자로 시작됨을 의미
    x$ 문자열의 종료를 표현하며 x 문자로 종료됨을 의미
    . 임의의 한 문자의 자리수를 의미
    (x) 그룹을 표현하며 x 를 그룹으로 처리함을 의미
    (x)(y) 그룹들의 집합을 표현하며 앞에서 순서대로 번호를 부여하여 관리하고 x, y 는 각 그룹의 데이터로 관리
    (x|y) or 를 표현하며 x 또는 y 문자가 존재함을 의미
    (x)(?:y) 그룹들의 집합에 대한 예외를 표현하며 그룹 집합으로 관리되지 않음을 의미
    [xy] 문자 선택을 표현하며 x 와 y 중에 하나를 의미
    [^xy] not 을 표현하며 x 및 y 를 제외한 문자를 의미
    [x-z] range를 표현하며 x ~ z 사이의 문자를 의미

    문자 집합과 특수 문자

    특정 문자들의 문자 집합을 사용할 때 여러 문자를 동시에 찾을 때 사용합니다.

    표현식 의미
    \^ escape 를 표현하며 ^ 를 문자로 사용함을 의미
    \b word boundary를 표현하며 문자와 공백사이의 문자를 의미
    \B non word boundary를 표현하며 문자와 공백사이가 아닌 문자를 의미
    \d digit 를 표현하며 숫자를 의미
    \D non digit 를 표현하며 숫자가 아닌 것을 의미
    \s space 를 표현하며 공백 문자를 의미
    \S non space를 표현하며 공백 문자가 아닌 것을 의미
    \t tab 을 표현하며 탭 문자를 의미
    \v vertical tab을 표현하며 수직 탭(?) 문자를 의미
    \w word 를 표현하며 알파벳 + 숫자 + _ 중의 한 문자임을 의미
    \W non word를 표현하며 알파벳 + 숫자 + _ 가 아닌 문자를 의미

    수량자

    패턴의 몇번 반복 되는지 나타내기 위해 패턴의 뒤에 수량자를 붙혀서 사용합니다.

    표현식 의미
    x+ 반복을 표현하며 x 문자가 한번 이상 반복됨을 의미
    x? 존재여부를 표현하며 x 문자가 존재할 수도, 존재하지 않을 수도 있음을 의미
    x* 반복여부를 표현하며 x 문자가 0번 또는 그 이상 반복됨을 의미
    x{n} 반복을 표현하며 x 문자가 n번 반복됨을 의미
    x{n,} 반복을 표현하며 x 문자가 n번 이상 반복됨을 의미
    x{n,m} 반복을 표현하며 x 문자가 최소 n번 이상 최대 m 번 이하로 반복됨을 의미

    패턴 변경자(Flag)

    패턴 구분자가 끝나면 그 뒤에 쓰는 것으로, 패턴에 일괄적으로 변경을 가할 때 사용합니다.

    Flag 의미
    g Global 의 표현하며 대상 문자열내에 모든 패턴들을 검색하는 것을 의미
    i Ignore case 를 표현하며 대상 문자열에 대해서 대/소문자를 식별하지 않는 것을 의미
    m Multi line을 표현하며 대상 문자열이 다중 라인의 문자열인 경우에도 검색하는 것을 의미
    블로그를 봐주셔서 감사합니다.
